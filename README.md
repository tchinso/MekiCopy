# MekiCopy 사용설명서

MekiCopy는 화면의 특정 영역을 지정한 뒤 OCR로 글자를 인식하고, 결과를 클립보드에 복사하는 도구입니다.

## 기본 사용 방법

### 1. 영역 지정
1. MekiCopy를 실행합니다.
2. `영역 지정` 버튼을 누릅니다.
3. 마우스로 글자가 있는 부분을 드래그해서 선택합니다.
4. 필요하면 가장자리를 드래그해서 영역을 미세 조정합니다.
5. `Enter` 키를 눌러 선택을 완료합니다.

### 2. 영역 설정
1. 메인 화면으로 돌아오면 방금 지정한 영역이 임시 영역으로 표시됩니다.
2. `영역 설정` 버튼을 눌러 실제 인식 영역으로 등록합니다.

### 3. 인식 후 복사
1. `인식 후 복사` 버튼을 누릅니다.
2. 지정한 영역의 텍스트가 인식됩니다.
3. 인식 결과가 클립보드에 자동으로 복사됩니다.

### 4. 붙여넣기
1. 메모장, 번역 사이트, 문서 편집기 등 원하는 곳을 엽니다.
2. `Ctrl + V`로 붙여넣습니다.

## 빠른 흐름 요약

`영역 지정` -> `Enter` -> `영역 설정` -> `인식 후 복사` -> 메모장이나 번역 사이트에 붙여넣기

## 텍스트 위치가 변하지 않는 경우

비주얼 노벨처럼 텍스트가 항상 같은 위치에 표시되는 경우에는 처음 한 번만 영역을 설정하면 됩니다.

1. 처음 한 번 `영역 지정` -> `Enter` -> `영역 설정`을 합니다.
2. 이후에는 같은 위치의 텍스트를 읽을 때마다 `인식 후 복사`만 누르면 됩니다.

즉, 화면 내 텍스트 위치가 고정되어 있다면 매번 영역을 다시 지정할 필요가 없습니다.

## 북마크 설정 방법

북마크를 저장해 두면 자주 쓰는 영역을 다시 쉽게 불러올 수 있습니다.

### 북마크 저장
1. `영역 지정` 버튼을 눌러 원하는 영역을 선택합니다.
2. 영역 선택 화면에서 필요하면 크기를 조정합니다.
3. `S` 키를 누릅니다.
4. 북마크 이름 입력창이 뜨면 원하는 이름을 입력합니다.
5. 저장이 완료되면 북마크가 저장됩니다.

### 저장한 북마크 불러오기
1. 메인 화면에서 `북마크 영역 불러오기` 버튼을 누릅니다.
2. 저장된 북마크 목록에서 원하는 항목을 선택합니다.
3. 선택한 북마크 영역이 임시 영역으로 불러와집니다.
4. `영역 설정` 버튼을 눌러 현재 인식 영역으로 적용합니다.
5. 이후 `인식 후 복사` 버튼을 눌러 바로 사용할 수 있습니다.

## 사용 팁

- 영역을 너무 작게 잡으면 인식이 잘 안 되거나 오류가 날 수 있습니다.
- 글자 주변에 여백을 조금 포함해서 잡으면 인식이 더 안정적일 수 있습니다.
- 같은 게임, 같은 창, 같은 자막 위치를 반복해서 읽는 경우 북마크를 저장해 두면 편합니다.
- 북마크를 불러온 뒤에도 필요하면 다시 영역을 지정해서 새로 설정할 수 있습니다.

## 추천 사용 예시

### 번역 사이트에 바로 붙여넣기
1. 게임 또는 프로그램 화면에서 텍스트 영역을 지정합니다.
2. `인식 후 복사`를 누릅니다.
3. 파파고, 구글 번역 등 번역 사이트 입력창에 붙여넣습니다.

### 메모장에 저장
1. `인식 후 복사`를 누릅니다.
2. 메모장을 열고 붙여넣습니다.
3. 필요한 문장을 계속 누적해서 저장합니다.
