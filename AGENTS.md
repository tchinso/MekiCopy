### MekiOCR에 대한 설명

MEKIOCR.md 파일에 있으니 읽어봐

### 사용자의 환경

사용자는 MS 스토어판 Python 3.13을 사용한다.

사용자는 2대 이상의 모니터를 사용하는 경우가 많다. 

모니터는 화면비율이나 해상도를 다르게 설정하는 경우가 많다. (예 : 한쪽은 세로형 모니터, 한쪽은 가로형 모니터)

그래서 각각의 화면 비율과 해상도를 각각 인식해야함.

### 니가 만든 것

1. 사용자가 MS 스토어판 Python 3.13만 설치했다는 가정하에 MekiOCR과 기타 필요한 파이썬 라이브러리들을 설치할 수 있도록 하는 Windows 배치 파일.

2. 사용자가 모니터 위에 특정 영역을 드래그하면 그 영역을 캡쳐하여 MekiOCR로 인식한 후 한 줄로 합쳐서 클립보드에 바로 복사
 (사용자가 복사 버튼을 한 번 더 누르는 일이 없도록, 바로 복사하고 복사되었습니다! 라는 메시지를 표시)

3. 사용자가 모니터 위에 특정 영역을 지정한 후에는 각 변을 드래그하여 캡쳐구역을 미세하게 조정할 수 있도록 한 후, 그 부분을 즐겨찾는 북마크로 등록할 수 있도록 하기.
 (북마크는 이름을 지정할 수 있고, 북마크는 개수가 여러개일 수 있으며, 북마크는 별도의 텍스트 파일 형태로 관리할 것.)

4. 사용자가 특정 북마크를 지정하면 해당 북마크가 가리키는 영역을 캡쳐하여 MekiOCR로 인식한 후 한 줄로 합쳐서 클립보드에 바로 복사
 (사용자가 복사 버튼을 한 번 더 누르는 일이 없도록, 바로 복사하고 복사되었습니다! 라는 메시지를 표시)

### 발생한 문제

복사를 시행했더니

Error initializing MeikiOCR: module 'onnxruntime' has no attribute 'set_default_logger_severity'

라는 문장만 복사가 되었어.
